Gulp Fiction

Framer

Features

Pros

  • Permite animaciones e interactividad compleja
  • Al ser código, permite iterar rápido.
  • Trabaja bien con Sketch
  • Ampliable con plugins
  • Buena documentación y comunidad
  • Permite reaprovechar código (hasta cierto punto)
  • Es javascipt

Contras

  • Un poco lento hasta que se le coge la mano.
  • Curva de aprendizaje medio-alta.
  • Es coffescript

¿Para qúe nos puede servir?

  • Prototipos con cierta complejidad en interacción
  • Se pueden crear flujos de navegación si se combina con View Controller

Flinto

Features

  • Permite hacer animaciones y flujos de navegación
  • Viewer y exportación de prototipos
  • Plugin de Sketch

Pros

  • Fácil y rápida comunicación con Sketch (plugin)
  • Documentación y tutoriales
  • Elementos reusables
  • Fácil de aprender / intuitivo

Contras

  • No sirve para prototipos muy complejos

¿Para qúe nos puede servir?

  • Prototipar de manera rápida
  • Transiciones y animaciones para componentes
